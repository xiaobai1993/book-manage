# 快速启动指南

## 前置要求

1. Node.js 16+ 和 npm
2. 后端服务已启动并运行在 `http://localhost:8080`

## 安装和启动

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

前端服务将在 `http://localhost:3000` 启动

### 3. 访问应用

在浏览器中打开 `http://localhost:3000`
