## 📋 新增需求流程

1. 在 `requirements/` 下创建新文件夹（如 `requirements/{功能名称}/`）
2. 创建 PRD 文档
3. 创建 API 文档（如需要）
4. 创建实施方案（如需要）
5. 创建其他相关文档（测试、迁移等）
6. 更新本索引文件
